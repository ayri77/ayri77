`Azure Synapse` · `Power BI` · `DAX` · `OData` · `SQL` · `Python`

### 📡 Distributed BI over 200+ ERP instances — UkrLandFarming PLC *(2011–2017)*
Architected a distributed BI system aggregating daily data from 200+ ERP (1C) instances. Built a centralised treasury and budget-planning system. Optimised heavy queries — **runtime cut from 12 h to 1.5 h** through parallelisation. Plus a 6 000-unit vehicle telemetry platform (GPS + sensors).
`SQL` · `Power BI` · `1C Enterprise` · `GPS / sensor data`

Knapsack-based optimisation for ad placement across 25+ TV channels; auction-logic, audience segmentation and forecasting framework.
`Mathematical modelling` · `SQL` · `DAX`

### 🏆 Shinkansen Customer Satisfaction Prediction — MIT Hackathon, **Rank #1**
End-to-end ML on tabular data: feature engineering 24 → 93 features, model comparison (CatBoost / LightGBM / XGBoost), AutoGluon AutoML. **96.245% test accuracy** + clear driver analysis.
`Pandas` · `CatBoost` · `LightGBM` · `XGBoost` · `AutoGluon`

### 🛰️ [Correction Polygon App](https://github.com/ayri77/correction_polygon_app) — End-to-end ML + API
ML coordinate correction service (SC63 → WGS84). Compared multiple regressors; RANSAC selected. Median error 0.3–0.8 m; IoU lifted from 0.5–0.85 to 0.94–0.99. Deployed FastAPI service on Render.
🔗 <https://correction-polygon-app.onrender.com/docs>
