## 🚀 Projects

### 🏆 Shinkansen Travel Experience Prediction — MIT Hackathon, **Rank #1 / 96.245% accuracy**
End-to-end satisfaction prediction. Feature engineering 24 → 93 features, model comparison (CatBoost / LightGBM / XGBoost), AutoGluon AutoML with feature selection, feature-importance analysis.
`CatBoost` · `LightGBM` · `XGBoost` · `AutoGluon` · `Stratified CV`

### 🛰️ [Correction Polygon App](https://github.com/ayri77/correction_polygon_app) — End-to-end ML + API
ML service for coordinate correction (SC63 → WGS84). Compared Linear / Ridge / RANSAC / MLP / XGB; RANSAC selected for outlier robustness. **Median error reduced from 6–13 m to ~0.3–0.8 m**; IoU **0.50–0.85 → 0.94–0.99**. Public API live on Render.
`scikit-learn` · `FastAPI` · `Shapely` · `Folium` · `Render`
🔗 <https://correction-polygon-app.onrender.com/docs>

### 🧠 [CIFAR10 Torch Classifier](https://github.com/ayri77/CIFAR10-Torch-Classifier) — Reproducible CV training framework
Modular PyTorch framework: MLP / custom CNN / ResNet18 / DenseNet121, Mixup & Cutout, deterministic runs, TensorBoard tracking, YAML configs, multiple optimisers, Kaggle submission workflow.
`PyTorch` · `TensorBoard` · `YAML`

### 🙂 Facial Emotion Detection — MIT Capstone
